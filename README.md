# Postify API

A backend-driven social media application built using Node.js, Express.js, and MongoDB.

## Features
- User Registration & Login
- JWT Authentication
- Create Posts
- Edit Posts
- Like/Unlike Posts
- Image Upload Support

## Tech Stack
- Node.js
- Express.js
- MongoDB
- Mongoose
- JWT
- Multer

## Run Locally

npm install
npm start
